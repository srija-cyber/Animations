Animation Types

1. Fade In/Out

Show/hide alert messages, modals, or tooltips using opacity.

2. Slide In/Out (X-axis)

Slide sidebar, toast, or drawer menu using translateX.

3. Slide In/Out (Y-axis)

Slide dropdown or filters from top using translateY.

4. Collapse/Expand

Show/hide FAQ or extra details in a section using height.

5. Scale/Zoom

Zoom card or image on hover using scale.

6. Rotate

Rotate refresh icon or toggle button using rotate.

7. Shake

Shake input field or login box on invalid entry using keyframes + translateX.

8. Flip (Y-axis)

Flip a card to show front/back using rotateY.

9. Skew

Apply skew effect on banner/image hover using skewX.

10. Bounce

Animate bouncing button or success alert using keyframes + translateY.

11. Zoom Out/Shrink

Shrink item on delete (like a note or task) using scale + opacity.

12. Grow Width/Height

Animate a progress bar or section using width or height.

13. Slide with Opacity

Slide in notifications or cards while fading in.

14. Fade + Expand

Open a detailed view or card with both opacity and height.

15. Flip with RotateY

Flip login/signup panel or flashcard using rotateY.
